MongoDB Configurations:
Create the collection names as follow  
1.	customer
2.	film

Maintain the Database name as follow
	dataset
Mongodb is configured to the port 27017 (Modification can be done in application.properties file)

Spring Boot Configuration:
Try to run the RestServiceApplication.java file and as the application is developed with the base of spring boot architecture it has in built tomcat and it is running in the default port 8080

To Run the application:

Import the project into the Eclipse with the STS flavor or Spring Tool Suite 3 

As it is a maven project try to run the maven update before running the application

This STS project has embedded tomcat to you can directly Right Click and Select Run as Spring boot App.
